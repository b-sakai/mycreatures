## 機能

* openGL によるタイトル画面の描画
* 図鑑のリスト表示機能
* 図鑑のツリー表示機能 (外部ライブラリ使用) 
* 種類の手動追加機能
* 種類の半自動追加機能( Wikipedia から)(開発中) 
* データベースによる保存
* 最近の登録表示機能
